**EDU2JOB** is an AI-based job role prediction system that recommends the most suitable career roles based on a user‚Äôs education, technical skills, certifications, work experience, projects, and career gaps.

---

## ‚≠ê Key Features
- 94% accurate ML model using Gradient Boosting (XGBoost)  
- Predicts **Top 3 job roles** (Gold / Silver / Bronze ranking)  
- Skill Gap Analysis with improvement suggestions  
- Prediction History with statistics  
- Model Insights: accuracy, feature importance, distribution analysis  
- Analytics Dashboard for trends and user insights  
- JWT-based secure login (password hashing included)  
- Admin Panel for managing users, predictions, and datasets  
- Clean and responsive Streamlit interface
